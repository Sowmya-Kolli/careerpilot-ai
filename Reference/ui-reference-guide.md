# HireTrack AI UI Reference

The file ui-prototype.mp4 contains the Google Stitch generated prototype.

Use it as visual inspiration.

Do not copy exactly.

Design requirements:

Theme:
- Light SaaS dashboard
- Purple/indigo accent
- Clean modern UI
- Rounded cards
- Soft shadows

Pages:

1. Landing Page
- Hero section
- Explain AI email tracking
- Features
- Start button

2. Dashboard
- Application statistics
- Recent job updates
- Status overview

3. Email Intelligence
- Email analysis panel
- AI generated summary
- Extract:
  Company
  Role
  Status
  Deadline
  Next action

4. Application Tracker
- Table format
- Status badges

5. Agent Activity
- Timeline
- Emails scanned
- Agent actions
- Security checks

Remove:
- Upgrade Pro
- Fake AI predictions
- Resume optimizer
